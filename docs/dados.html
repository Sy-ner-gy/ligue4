Dados



#Legendas


▪️[user] : dados gerados pela interação do player (Saídas no Controller - Manipulador;

▪️[controller] : dados gerados internamente pelo Controller - Gerador para serem usados por toda a aplicação;

▪️Entradas-Dados: dados que serão necessários para eventos no View ou no Model

▪️Entradas/Saídas-Mídias: mídias que deverão ser armazenadas no cache e localstorage

▪️Saídas: dados gerados através da interação do user

▪️origin: dado gerado no Controller-Game Rules, que será usado pelo Controller-Gerador para gerar todos os demais dados




#Observações


▪️Ao fazer alteração nessa documentação, por favor atualize a versão do mesmo, mudando o seu nome e fazendo o upload para o Trello

▪️Os nomes dos dados são somente sugestivos, podendo o desenvolvedor mudá-lo, desde que o documente de forma correta.
